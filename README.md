# Kelompok-1-Project-Akhir-PBP
Deskripsi
Resto Delight adalah tempat kuliner yang menyajikan berbagai hidangan lezat dengan bahan-bahan segar dan pelayanan yang hangat. Kami berkomitmen untuk memberikan pengalaman bersantap yang menyenangkan bagi setiap pelanggan.

Fitur-fitur utama
Multi-level Authentication
•	Admin: Full access semua fitur
•	Kasir: Transaksi dan manajemen menu
•	Pelanggan: Pesan menu dan lihat riwayat

Dynamic Menu Management
•	CRUD operasi untuk menu
•	Kategorisasi otomatis
•	Upload gambar menu
•	Sistem stok real-time

Shopping Cart System
•	Session-based cart
•	Quantity validation
•	Real-time total calculation
•	Multiple payment methods

Real-time Notifications
•	AJAX polling system
•	Toast notifications
•	Badge counters
•	Database triggers untuk auto-notifikasi

QR Code Integration
•	QR Code untuk verifikasi pesanan
•	External API integration
•	Data encryption untuk keamanan
•	Scan simulation untuk testing

Reporting System
•	Chart.js integration
•	Daily/weekly/monthly reports
•	Best selling items analysis
•	Revenue tracking
•	Export functionality
